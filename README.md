# Hamptons Home Management Website

This repository is a app for Hamptons Home Management Website, powered by [Next.js](https://nextjs.org/) & [Payload CMS](https://payloadcms.com/)

## Installation

### Using nvm

- Run `nvm install && nvm use`
- Install [pnpm@^10](https://pnpm.io/motivation)
- Run `pnpm install`

### Using asdf

- Run `asdf install`
- Run `pnpm install`

## Running

- Copy `.env.example` to `.env.local`
- Run `docker compose up -d`
- Run `pnpm run dev`

Exposed local services:

- [App](http://localhost:3000/)
- [MongoDB](http://localhost:27017/)
